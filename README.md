# Christian Coenen Portfolio

This repository contains the source code for Christian Coenen's portfolio website, built with React. The portfolio is a modern, minimalistic website showcasing projects and qualifications.

## Technologies
- **Front-end:** React, JavaScript (ES6+)
- **Styling:** CSS
- **Build tool:** Vite
